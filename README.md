# Senior Tech Guide

**Senior Tech Guide** es una aplicación Android nativa diseñada específicamente para acortar la brecha digital en las personas mayores. El objetivo principal es proporcionar un entorno intuitivo, guiado y, sobre todo, seguro, donde los usuarios senior puedan aprender y practicar el uso de las aplicaciones cotidianas más comunes sin miedo a cometer errores en el mundo real.

---

## Características Principales

La aplicación se divide en dos enfoques pedagógicos clave a través de una interfaz limpia y accesible:

### 1. Módulo de Misiones (Aprendizaje Guiado)
* **Objetivos Diarios:** Propone retos específicos al usuario (ej. *"Enviar un WhatsApp"*, *"Hacer un Bizum"* o *"Pedir Cita Médica"*).
* **Sistema de Recordatorios:** Alertas visuales amigables que motivan al usuario a retomar la práctica si lleva tiempo sin interactuar (ej. *¡Hace 48 horas que no practicas!*).
* **Contador de Progreso:** Indicadores claros (`0 / 5`) para celebrar los logros del día.

### 2. Modo Seguro (Exploración Libre)
* **Entorno Sandbox:** Un espacio de simulación interactiva donde las tarjetas de las aplicaciones tienen una distribución geométrica y simétrica perfecta (reja unificada a `190dp` de altura).
* **Cero Riesgos:** Los usuarios pueden interactuar con interfaces clonadas de **WhatsApp, Bizum, Salud, Gmail y Google Maps** con la tranquilidad de que no se enviará dinero real ni mensajes accidentales.
* **Botón de Ayuda Flotante:** Un acceso rápido e identificable en todo momento (`💡 Pista / Ayuda`) para resolver dudas en tiempo de ejecución.
